Наши работы

Робот удаленного присутствия

Для воплощения дизайнерских эскизов в жизнь была реализована математическая модель конструкции робота. Разработана конструкция корпуса, головы, зарядного устройства. Построена модель сложных поверхностей, далее разбитая на отдельные детали. Разработаны места соединения деталей, схема сборки - разборки. Весь конструктив смоделирован с учетом возможности изготовления различными технологиями производства. Дизайн разработан компанией «Астраросса дизайн». Работа выполнена по заказу «Ашманов и партнеры».

Робот изготовлен, прошел испытания.

Математическая модель факела Сочи 2014

По заказу «Астраросса дизайн» разработана математическая модель корпуса, с учетом возможности соединения и сборки деталей различными вариантами, с уклоном под изготовление литьем. В результате разработки рассмотрено множество вариантов компоновки, выбраны наиболее технологичные и быстрые способы соединения, с использованием неодимовых магнитов. На базе данной модели построена технологическая и производственная цепочка, по которой начато производство факелов.

Разработка дизайна корпуса робота-куба

Разработана серия вариантов конструкции корпуса для перспективного устройства - робота куба с функциями удаленного управления различными электронными устройствами в быту. Работа выполнена по заказу «Кубик роботикс».

Антенна

В рамках аванпроекта разработана компоновка лабораторного стенда для проведения физических исследований. Работа выполнена по заказу МГТУ им. Н.Э.Баумана.

Доменная печь

Металлургическая печь служит для получения чугуна и является отправной точкой в цепи получения высококачественной конструкционной стали. В рамках работы выполнена детальная графическая модель конструкции доменного цеха, наглядно продемонстрированы основные узлы.

Аппараты для лечения повреждённых суставов

Реанимация советской разработки аппаратов для лечения суставов. В рамках работы воссозданы математические модели и конструкция устройств, введены улучшения, разработан комплект конструкторской документации и передан на производство. Аппараты технологичны в производстве и обладают минимальной себестоимостью по сравнению с мировыми аналогами. Работа выполнена по заказу «Минерва-кэпитал».
